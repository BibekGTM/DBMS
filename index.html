DBMS.

What is DBMS?

DBMS stands for Database Management System. It is software that interacts with end users, applications, and the database itself to capture and analyze data.

Terminologies

Data

In the context of a Database Management System (DBMS), "data" refers to the information that is stored, processed, and managed by the system. This data can be structured in various ways depending on the type of database and its design.

Information

Information refers to knowledge communicated or received concerning a particular fact or circumstance. In computing, it is data processed or organized for analysis or use.

Meta Data

Metadata is data that describes other data. It provides information about a certain item's content, quality, condition, or other characteristics.

Data Dictionary

A data dictionary is a centralized repository of information about data, such as its meaning, relationships to other data, origin, usage, and format.

Database

A database is an organized collection of data, typically stored and accessed electronically from a computer system. It is designed to manage large amounts of data efficiently.

Table

In a database, a table is a collection of related data organized in rows and columns. It represents a logical structure used to store and manage data in a relational database management system (RDBMS).

Tuple

In the context of databases, a tuple (or record) is a row in a table that represents a single instance of data. It contains a set of field values that uniquely identify it within the table.

Field

A field is a single piece of data in a database record, representing an attribute of the entity the record describes. It corresponds to a column in a table.

Domain

In database terminology, a domain defines the set of possible values that a field in a table can contain. It specifies the data type, format, and constraints of the data.

Constraint

A constraint in a database is a rule or condition that limits the values that can be placed in a column or a combination of columns. It ensures data integrity by enforcing rules on the data.

Key

In database management, a key is a field or set of fields that uniquely identifies a record in a table. It is used to enforce uniqueness and to establish relationships between tables.
